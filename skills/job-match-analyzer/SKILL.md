# Job Match Analyzer Skill

This skill analyzes job postings and matches them against a candidate's background, skills, and experience.

## Purpose

To provide data-driven insights on how well a candidate matches a specific job opportunity, with actionable recommendations for tailoring application materials.

## Reference Materials

This skill uses the following reference data:

1. **experience_inventory.csv** - Complete work history with roles, responsibilities, and achievements
2. **skills_taxonomy.csv** - Comprehensive list of technical and soft skills
3. **corporate_translation.csv** - Mapping of education terminology to corporate equivalents
4. **achievement_library.csv** - Quantified achievements and success stories

## Analysis Approach

The skill analyzes:
- Hard skills match (technical capabilities)
- Transferable skills (how education experience translates)
- Soft skills and cultural fit
- Experience level appropriateness
- Education and qualifications

## Output

Provides:
- Match score (0-100)
- Key strengths and gaps
- Resume tailoring suggestions
- Cover letter guidance
- Red flags and research recommendations
